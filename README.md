# Wholesale Dai

Wholesale Dai is a closed e-commerce platform built with React.js, designed for wholesale transactions. This platform allows registered users to browse, place orders, and manage their inventory efficiently.

## Features

- **User Authentication**: Secure login and registration for wholesale buyers.
- **Product Catalog**: Browse products with detailed descriptions, pricing, and images.
- **Order Management**: Easy placement, tracking, and management of orders.
- **Admin Dashboard**: Admin panel for managing products, orders, and user accounts.
- **Responsive Design**: Fully responsive UI for an optimal experience on desktops and mobile devices.
- **Secure Payments**: Integrated payment system for handling transactions.
